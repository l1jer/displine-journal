```html
<a href='../xxx.html'> </a>中:
  '../' -> 上层目录(../news/news.html)
	'./' -> 当前目录, 不用加(./banner.png)
  '/' -> 根目录(/img/banner.png)
  
'<!-- -->' -> HTML 注释
'{{-- --}}' -> 后台注释
```

```html
<form>
  <fieldset>
  	<legend>xxx</legend>
  	<input type="file" name="image[]" accept="image/png" multiple>
    <!-- 'image[]'提交上传多个文件到数组, 
		'accept'限定上传格式,
		'multiple'允许多个上传 -->
	</fieldset>
	<button type="submit">Submit</button>
</form>
```
